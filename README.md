#symptom checker bot:- 

A simple Python Chatbot that provides basic advice based on input symptoms. This console-based tool helps tips or emergency warnings based on common symptoms 

#features:-

1) input multiple symptoms separated by commas.
2) provides appropriate response and basic advice for a list of common symptoms.
3) special handling and urgent warnings for "chest pain"and "difficulty breathing".
4) user friendly prompts and clear instructions.

#Requirments 

1)Python3.x
2)No external libraries needed
